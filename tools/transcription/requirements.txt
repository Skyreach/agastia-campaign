# Audio Transcription Tool - Python Dependencies
# Tested: 2026-01-25
# Python: 3.10+
# Platform: WSL2 Ubuntu / Linux / macOS

# ============================================================
# INSTALLATION
# ============================================================
#
# 1. Install system dependency:
#    sudo apt install ffmpeg
#
# 2. Create virtual environment (recommended):
#    python3 -m venv ~/.venvs/transcription
#    source ~/.venvs/transcription/bin/activate
#
# 3. Install Python dependencies:
#    pip install -r requirements.txt
#
# 4. Configure HuggingFace token (for speaker diarization):
#    - Get token: https://huggingface.co/settings/tokens
#    - Accept terms: https://huggingface.co/pyannote/speaker-diarization-3.1
#    - Save: echo "YOUR_TOKEN" > config/hf_token.txt
#
# 5. Run:
#    cd scripts && python3 process_session_transcript.py
#
# ============================================================
